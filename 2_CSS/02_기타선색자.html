3. 동위 선택자

동위관계(같은 레벨)에 있는 뒤에 위치한 특정 요소를 선택할 때 사용

    * a요소 뒤에 있는 b요소 "하나만"을 선택하려할때
    a+b {
        스타일속성:값;
    }    
    
    * a요소 뒤에 있는 "모든" b요소 다 선택
    a~b {
        스타일속성:값;
    }

5. 상태 선택자

요소의 상태에 따라서 선택되는 선택자

    1) checked(체크된) 상태의 요소를
    선택자 : checked{
        스타일 속성 : 값;
    }
    

    2) 초점(focus)이 맞춰진 input요소 선택
    선택자 : focus{
        스타일속성:값;
    }

    3) 활성화된 요소만을 선택
    선택자:enabled{

    4) 비활성화된 요소만을 선택
    선택자:disabled{
        스타일속성:값;
