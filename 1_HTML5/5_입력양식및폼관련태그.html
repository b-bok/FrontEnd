라디오버튼과 체크박스 관련 input태그의 타입

라디오버튼 (type ="radio")

- 제시한 여러개의 값들 중 한개만 선택가능
(name 속성값이 똑같은 것들끼리 하나의 그룹으로 지어줌) - 선택된 값이 넘어감
- 선택된 값을 넘기고자 한다면, 선택했을 때 어떤 값을 넘길 건지 value 값을 명시해둬야함 -

성별 :

체크박스 (type = "checkbox")

- 제시한 여러개의 값들 중에서 여러개 선택 가능할 때
- 선택된 값(checked)이 넘어감
- 넘기고자 하는 value값 명시해둬야함

select태그와 option태그

- 사용자에게 직접 입력받는 게 아니고, 여러 옵션들 중(목록)에서 선택 가능하게
- 값 전달시 selected된 값이 넘어감
- 각각의 option에 value 속성값 명시했을 경우 value값이 전달됨
- 명시 하지 않았을 경우, 기본적으로 option태그의 시작태그와 종료태그 사이값이 넘어감
