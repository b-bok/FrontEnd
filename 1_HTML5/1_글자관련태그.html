*문단을 나누는 태그(p, pre)

문단 영역을 나누는 p태그와 pre태그가 있다 p태그는 문단 영역을 나누는 태그지만 줄바꿈을 하고 싶다면
별도의 태그를 기술해야됨 뿐만 아니라 한개의 공백만 표시하기 때문에 추가적인 공백을 넣고자     한다면 특정 구문을 작성해야한다. 

pre태그는 여러칸     띄어쓰기 적용됨
줄바꿈도 알아서 적용됨

* 기타 글자 관련 태그

b : 글자를 굵게 하는 태그
strong : 글자를 굵게 표시하는 태그
em : 글자를 기울이는 태그
mark : 형광펜 효과를 나태내는 태그
u : 글자 밑에 밑줄을 그어주는 태그
s : 글자에 취소선을 넣는 태그
small : 글자를 작게 표시하는 태그
기본 글자에 아래첨자를 나타내는 태그와 윗첨자를 나타내는 태그
IoT란 각종 사물에 센서와 통신 기능을 내장해 인터넷에 연결하는 기능
