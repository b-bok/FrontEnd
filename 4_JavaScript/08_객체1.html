객체

* 객체 선언 및 호출

객체는 중괄호{}를 사용해서 생성하고,
중괄호 안에 이 객체에 필요한 속성을 정의한다.
속성 : 속성값의 형태로 정의함 (키:밸류)
속성값에는 모든 자료형 값 올 수 있음 

        var 변수명 = {

            속성명 : 속성값,
            속성명 : 속성값,
            속성명 : 속성값
            ...
        };

* 객체 생성 특이케이스

* 객체에서의 반복문

객체가 가지고 있는 모든 속성들에 순차적으로 접근하고자 한다면 반복문 제시 가능
단, 단순 for loop문으로는 사용 불가능하고 for in문만 사용 가능함

* 객체의 메소드 속성

객체의 속성 중에 함수 자료형인 속성을 메소드라고 부른다.

* in과 with키워드

in : 객체 내부에 해당 속성이 있는지 확인해주는 키워드
with : 객체명을 생략할 수있음, 코드 줄일 수 있음!
