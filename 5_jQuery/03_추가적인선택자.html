* 속성 선택자

선택자[속성] : 특정 속성을 가지고 있는 모든 요소 선택
선택자[속성 =속성값] : 속성안의 값이 특징값과 일치하는 모든 요소 선택
선택자[속성~=속성값] : 속성안의 값에 특정값을 단어로 포함하는 모든 요소 선택
선택자[속성^=속성값] : 속성 안의 값이 특정값으로 "시작"하는 모든 요소 선택
선택자[속성$=속성값] : 속성안의 값이 특정값으로 "끝"나는 모든 요소 선태
선택자[속성*=속성값] : 속성안의 값에 특정값을 "포함"하는 모든 요소 선택
