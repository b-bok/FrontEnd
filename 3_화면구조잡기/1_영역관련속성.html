영역관련 속성

1) content영역 : 요소내의 내용물 영역
2) border영역 : 요소의 테두리 영역
3) padding영역 : content와 border사이의 영역(여백)
4) margin영역 : 요소의 바깥쪽 영역
div의 테두리까지 포함한 가로길이
= width 값 + 양쪽 padding 값 + 양쪽 border 값
= 300px + (10*2)px + (20*2)px
= 380px
div의 테두리까지 포함한 세로 길이
= height값 + 상하padding값 + 상하border값
= 150px + (10*2)px + (20*2)px
= 210px
